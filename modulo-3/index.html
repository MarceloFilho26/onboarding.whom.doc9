Como criar uma concessão de acesso

Para liberar o uso da plataforma Whom para colaboradores, é necessário criar uma concessão de acesso no painel administrativo.

⚠️ Lembre-se: O pré-requisito para criar uma concessão é ter, no mínimo, um grupo de restrições criado.

Passo a passo:

  1. Acesse o Painel do Admin.
  2. Clique em Nova Solicitação.
  3. Selecione Concessão de Acesso ou Concessão de acesso a Múltiplos Certificados (se você tiver 1 ou mais certificados).
  4. Preencha o formulário, que é dividido em três blocos:
    • Bloco 1: selecione o grupo de restrição e o grupo de usuários (opcional).
    • Bloco 2: escolha o(s) certificado(s), defina a validade da concessão (data e horário) e insira os dados do(s) usuário(s): e-mail, nome completo e CPF.
      É possível incluir mais de um usuário na mesma solicitação, preenchendo os dados e clicando no botão "Adicionar".
    • Bloco 3: selecione os sistemas que o usuário poderá acessar.
  5. Finalize a solicitação clicando em Conceder Acesso para registrar a concessão.
➡️ Dica: Após a concessão, o usuário receberá automaticamente um e-mail com orientações para baixar e autenticar a extensão Whom.

Temas relacionados

🔹 Também é possível cadastrar os usuários de um mesmo grupo de restrições através da planilha

Após o cadastro feito pelo administrador, o colaborador recebe um e-mail automático com o link para download da extensão Whom.

Passos para o primeiro acesso:

  1. Baixar a extensão no navegador (Google Chrome ou Microsoft Edge).
  2. Fixar a extensão para que fique sempre visível na barra do navegador.
  3. Abrir a extensão e digitar o e-mail cadastrado.
  4. Inserir o código de verificação recebido por e-mail (copiar e colar).
  5. A extensão realizará o login automático, liberando os sistemas definidos pelo administrador.
➡️ A partir desse momento, todo o acesso aos sistemas será feito diretamente pela extensão Whom, de forma prática e segura.
